# AI Drone Disaster Management System

Real-Time AI-Based Drone Disaster Detection and Survivor Identification System Using YOLOv8.

## Features

- Human (Survivor) Detection using YOLOv8
- Fire and Smoke Detection (Optional Custom Model)
- Flood Detection using HSV color segmentation
- Works Offline
- Virtual Camera Support (ApowerMirror)
- Optimized for Real-Time Performance

## Installation

pip install -r requirements.txt

## Run

python main.py

## Requirements

- Python 3.9+
- Webcam or Virtual Camera Feed
- YOLOv8 Model
